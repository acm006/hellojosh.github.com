Chevrolet Route 66

http://chevroletroute66.msn.com

Built for MSN

MSN is working with Chevrolet to come up with a campaign for the Super Bowl. Route 66 is a international contest to see which user can share the most videos. The sharer with the most shares wins a huge prize.

ePrize was used to store all of the campaign's information including video views, user registration, share count and more. Brightcove was used to store and serve the videos to the site. The ePrize and Brightcove APIs were mashed together to serve the content to the users of the site. The site ran in a number of different countries and 8 different languages were used in the site for international users.

The interface of the site is powered by jQuery, SammyJS and AJAX. The entire site is responsive and it optimized to work well on desktop browsers, tablet and mobile devices.

Trending Tracker

http://tracker.springbox.com

Built for Springbox

Trending Tracker is a one-stop spot for our clients and friends to quickly check the pulse of their web presence. Social feeds are cosumed everyday and the data is stored in a data base. When the user comes to the site, he/she is presented with an easy way to see all of the social statistics for the previous day and their overall followers, likes, tips, check-ins, etc.

Mobile was the target when the concept for Trending Tracker was thought of. The idea was that the user could check their social statistics quickly where ever he/she was. The site has a responsive design that makes it easy to use on any mobile device.

A custom CMS tool was built for the project so the we could easily add and edit accounts.

Public data is retrieved from these sources: Alexa, Twitter, Facebook, Foursquare, Gowalla.

Fox Trial Finder

http://foxtrialfinder.org

Built for Michael J. Fox Foundation

Built for Michael J. Fox Foundation. Fox Trial Finder is a web application that allows Parkinson's paitents, their families or their friends to find Parkinson's drug trials. The system also allows trial teams that run the drug trials to contact the user for more information.

The system automitacally matches users to drug trials based on the patients profile. The system also sends a weekly email to all users informing them of their newest trial matches. The Fox Trial Finder also has an in-site message system that allows patients and trial teams to easily communicate.

The Fox Trial Finder has dashboard that allows super administrators and trial teams to fully manage trials which includes: creating, editing, deleting and adding new locations to. The system also allows super administrators and trial teams to search for volunteers and to see volunteers that are interested in their trials.

Patients also have a dashboarb that allows them to edit their profile, see trial matches and see messages from and to trial teams.
